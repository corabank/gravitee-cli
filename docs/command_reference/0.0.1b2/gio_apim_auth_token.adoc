= gio apim auth token

sign in with token

[source,shell]
----
$ gio apim auth token [OPTIONS] [TOKEN]
----

Sign in with personal access token

if token start with `env:` the token will retrieve from environment variable. i.e: `env:TOKEN`.

== Options

[cols="2a*"]

|===

2+| *--name TEXT*

|Description | token name

|===
